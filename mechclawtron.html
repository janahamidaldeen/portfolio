Mech-Claw-Tron – LEGO EV3 Claw Machine

As part of a robotics project, my team and I designed and built Mech-Claw-Tron, a fully functional claw machine using the LEGO EV3 robotics kit and RobotC programming. Our goal was to create a nostalgic, interactive structure that provided entertainment using commonly found objects. The project integrated mechanical design and programming skills from MTE100 and MTE121.

We replicated real-world claw machine functionality by designing a 3-axis system (shown above) with a functional mechanical claw, a prize chute, and a structured frame, all built from LEGO components. RobotC was used to program precise movement and task flow, ensuring smooth operation.

This project strengthened my skills in robotics, embedded systems, and collaborative problem-solving, while showcasing my ability to bring creative engineering concepts to life.
